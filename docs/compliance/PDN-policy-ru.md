# Политика обработки персональных данных (ЗРУ-547)

**Статус:** documented draft for sign-off §8.3  
**Владелец:** Legal / DPO  
**Дата:** 2026-07-01

## 1. Оператор

ООО «Jomboy Lavka», Республика Узбекистан.

## 2. Цели обработки

- Регистрация и аутентификация пользователей
- Исполнение договора доставки продуктов
- Платежи (Payme, Click — redirect-only, PCI SAQ-A)
- Поддержка клиентов и возвраты
- Маркетинг (только при отдельном согласии)

## 3. Категории ПДн

Телефон, ФИО, адрес доставки, история заказов, платёжные метаданные (без PAN).

## 4. Права субъекта

Доступ, исправление, удаление — через приложение или support@jomboy.uz.

## 5. Хранение

PostgreSQL (Uztelecom), резервные копии MinIO WORM, срок — 5 лет после последнего заказа.

## 6. Реализация в продукте

- Экран согласия в Flutter onboarding (`apps/customer-flutter`)
- Audit log Admin API → `admin_audit_log` (admin-bff)
